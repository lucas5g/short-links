# Short Links 

## Descrição 

Short Links é uma api desenvolvida em Node.js que permite aos usuários encurtarem URLs longas em URLs mais curtas e fáceis de compartilhar. 

## Tecnologias 
* Node.js
* TypeScript
* Redis
* PostgreSQL
* Docker
## Instalação 
```
docker compose up -d 
npm i
```

## Execução 
```
npm run dev
```
